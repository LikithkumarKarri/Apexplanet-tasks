What is HTML?

HTML stands for HyperText Markup Language. It is used to structure content on the web.

What does CSS do?

CSS (Cascading Style Sheets) controls the look and layout of HTML elements on a webpage.

What is JavaScript used for?

JavaScript is used to make webpages interactive, like buttons, sliders, and dynamic content.
